# 🌤️ Weather App

A desktop weather application built using **Python**, **PyQt5**, and the **OpenWeather API**.

## Features

- Search weather by city
- Display current temperature
- Weather description
- Weather emojis
- Error handling
- Desktop GUI

## Technologies

- Python
- PyQt5
- Requests
- OpenWeather API
- python-dotenv

## Installation

```bash
pip install -r requirements.txt
python mainWeather.py
```
